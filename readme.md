# Sort Files

> **Sorting files by file format**

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements.txt
```

## Usage

> open the sort.json and change path.

```json
{
	"data": [
	{
		"from": "C:/Users/...",
		"to": "C:/Users/...",
		"format": ["mp3"]
	}
	]
}
```
